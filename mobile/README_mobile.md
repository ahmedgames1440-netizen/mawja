# موجة | Mawja 📡

تطبيق تحليل وتحسين شبكات WiFi — يعمل مع جميع شركات الاتصالات
(STC، موبايلي، زين، سلام...) لأنه يتعامل مع الشبكة نفسها، لا مع المزود.

## المزايا
- 🛰️ **رادار حي**: قوة الإشارة والبنق لحظياً بواجهة رادار متحركة
- ⚡ **اختبار سرعة حقيقي**: تحميل ورفع عبر خوادم Cloudflare + بنق وتذبذب
- 🏆 **أفضل موقع للمودم**: امشِ في البيت، قِس كل غرفة، ويرتبها التطبيق بالنقاط
- 📊 **تحليل القنوات** (أندرويد): كشف الازدحام واقتراح القناة الأفضل
- 🌐 **سباق DNS**: يحدد أسرع خادم DNS لشبكتك
- 🌙 واجهة عصرية داكنة + عربي/إنجليزي مع RTL كامل

## خطوات البناء
```bash
flutter create --org com.ahmed.mawja mawja_base   # هيكل المنصات
# انسخ lib/ و pubspec.yaml من هذا المشروع فوق mawja_base
flutter pub get
flutter run                    # تجربة مباشرة
flutter build apk --release    # أندرويد
flutter build ios              # iOS (على ماكبوكك مع Xcode 15.2)
```
- أضف صلاحيات أندرويد من `AndroidManifest_permissions.xml`
- أضف مفاتيح iOS من `ios_Info_plist_additions.txt` وفعّل "Access WiFi Information"
- أو استخدم `.github/workflows/build.yml` للبناء السحابي للمنصتين

## ملاحظة تقنية صادقة
لا يمكن لأي تطبيق "تسريع" الراوتر برمجياً — Apple وGoogle لا تسمحان بالتحكم
في أجهزة الراوتر. التسريع الحقيقي يأتي من: الموقع الصحيح، القناة الأقل ازدحاماً،
وDNS أسرع — وهذا بالضبط ما يقدمه موجة. وعلى iOS تحديداً تمنع Apple قراءة
قوة إشارة WiFi، لذا يعتمد التطبيق هناك على قياس السرعة والاستجابة (بنفس الدقة العملية).
